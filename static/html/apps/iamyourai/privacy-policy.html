I am your AI 개인정보 처리방침

"I am your AI"(이하 "확장프로그램")은(는) 이용자의 개인정보를 중요하게 생각하며, 관련 법령을 준수하고 이용자의 권익을 보호하기 위해 본 개인정보 처리방침을 공개합니다.
본 확장프로그램은 별도의 백엔드 서버 없이 동작하며, 사용자가 생성한 대부분의 정보(요약/채팅 히스토리/저장된 요소/실행 그래프/설정 등)는 사용자의 로컬 환경(브라우저 저장소)에만 저장됩니다.

1. 개인정보의 처리 목적

확장프로그램은 다음 목적을 위하여 개인정보를 처리합니다. 처리되는 정보는 아래 목적 이외의 용도로 이용되지 않으며, 목적이 변경되는 경우 관련 법령에 따라 필요한 조치를 이행합니다.

  • 서비스 제공 및 기능 수행: 웹페이지 본문 추출/요약, 사이드패널 채팅, 번역, 선택 텍스트 변환, 저장된 요소 및 실행 그래프 기반 자동화 기능 제공
  • 설정 및 사용자 경험 유지: AI 제공자/모델 선택, 폰트 크기, 요약 길이, 채팅 저장 개수, 링크 열기 방식 등 사용자 설정 저장
  • 문의 대응 해당 시: 이용자가 문의 채널을 통해 연락하는 경우, 문의 처리 및 답변 제공

중요 안내(로컬 저장 원칙)
확장프로그램에서 사용자가 생성한 대부분의 정보(요약 결과, 채팅 히스토리, 저장된 요소, 실행 그래프, 설정값 등)는 사용자의 브라우저 로컬 저장소(예: chrome.storage 등)에만 저장되며, 확장프로그램 개발자가 운영하는 별도 서버로 전송·저장하지 않습니다.

2. 처리하는 개인정보의 항목

확장프로그램은 원칙적으로 최소한의 정보만을 처리합니다. 아래 항목은 실제 사용 설정에 따라 일부만 처리될 수 있습니다.

구분 처리 항목 수집/생성 경로 비고
설정 정보 AI 제공자/모델 선택값, 폰트 크기, 요약 제한 글자수, 채팅 저장 개수, 링크 클릭 동작 등 이용자 입력/선택 로컬 저장
API 키 OpenAI/Google/Anthropic 등 AI 제공자 API 키 이용자 직접 입력 로컬 저장. 이용자가 선택한 AI 제공자 호출에만 사용
사용자 생성 콘텐츠 채팅 메시지, 요약 결과(마크다운), 저장된 요소(이름/선택자/URL), 실행 그래프(노드/연결/설정) 이용자 입력 또는 사용 중 생성 로컬 저장
페이지 컨텍스트 현재 페이지 본문 텍스트(예: "@페이지" 기능), 사용자가 선택한 텍스트, URL 등 이용자의 기능 실행 시 확장프로그램이 웹페이지에서 추출 AI 요청 시 이용자가 선택한 제공자에게 전송될 수 있음
문의 처리 이메일 주소, 문의 내용 등(이용자가 제공한 범위) 이용자가 문의 채널로 제공 해당 시

※ 확장프로그램은 원칙적으로 주민등록번호 등 고유식별정보 및 민감정보의 처리를 의도하지 않습니다. 다만 이용자가 웹페이지 본문/선택 텍스트에 민감정보를 포함한 내용을 직접 입력·전송할 수 있으므로, 해당 전송 여부는 이용자가 기능 사용 시 결정하게 됩니다.

3. 개인정보의 처리 및 보유 기간

확장프로그램은 개인정보를 목적 달성에 필요한 기간 동안만 보유·이용하며, 원칙적으로 로컬 저장소에 보관됩니다.

  • 설정 정보 / API 키 / 채팅·요약·저장 데이터: 이용자가 삭제하거나 확장프로그램을 제거(언인스톨)할 때까지
  • 채팅 히스토리: 이용자가 설정한 저장 개수 정책에 따라 오래된 항목부터 자동 삭제될 수 있음
  • 문의 처리 정보 해당 시: 문의 처리 완료 후 내부 기준에 따라 즉시 또는 일정 기간 보관 후 삭제

4. 개인정보의 파기 절차 및 방법

확장프로그램은 개인정보가 불필요하게 되었을 때 지체 없이 파기합니다.

  • 파기 절차: 이용자의 삭제 동작(기록 삭제/초기화) 또는 확장프로그램 제거 시 로컬 저장 데이터가 삭제됩니다.
  • 파기 방법: 브라우저 저장소에서 해당 데이터를 삭제(전자적 삭제)합니다.

※ 이용자가 브라우저 동기화 기능을 사용하는 경우, 동기화 정책에 따라 일부 설정값이 계정에 연동될 수 있으므로, 동기화 해제/관리 기능은 브라우저(Chrome) 설정에서 조정할 수 있습니다.

5. 개인정보의 제3자 제공

확장프로그램은 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다.

다만, 이용자가 AI 기능(요약/채팅/번역/텍스트 변환/자동화 중 AI 노드 등)을 실행하는 경우, 이용자가 선택한 AI 제공자에게 요청이 전송되며, 이 과정에서 아래 정보가 포함될 수 있습니다.

제공받는 자 제공 목적 제공 항목 보유·이용 기간
OpenAI / Google / Anthropic 등 이용자가 선택한 AI 제공자 AI 응답 생성(요약/번역/대화/텍스트 변환/분석 등) 이용자 입력 프롬프트, 선택 텍스트, 페이지 본문("@페이지" 사용 시), URL 등 이용자가 기능 실행 시 포함한 데이터 AI 제공자의 정책에 따름

중요
확장프로그램 개발자는 이용자의 요청 데이터를 자체 서버로 중계하거나 저장하지 않습니다. AI 요청은 이용자가 입력한 API 키를 사용하여 이용자의 브라우저에서 AI 제공자로 직접 전송됩니다.

6. 개인정보의 국외 이전

이용자가 선택한 AI 제공자(OpenAI/Google/Anthropic 등)의 서버 위치에 따라, AI 기능 실행 시 요청 데이터가 국외로 이전(전송)될 수 있습니다.

이전 항목 이전 국가/수신자 이전 시기·방법 이용 목적 거부 방법·효과
이용자 입력 프롬프트, 선택 텍스트, 페이지 본문("@페이지" 사용 시), URL 등 이용자가 선택한 AI 제공자(각 제공자의 서버 운영 국가) AI 기능 실행 시점에 HTTPS API 통신으로 전송 AI 응답 생성 및 서비스 제공 거부 방법: AI 기능을 사용하지 않거나, "@페이지" 기능을 사용하지 않고 프롬프트를 최소화하여 전송 범위를 조절할 수 있습니다.
거부 효과: AI 기반 요약/번역/채팅/변환/AI 노드 기능의 이용이 제한됩니다.

※ AI 제공자가 수신한 데이터의 처리(보관/삭제/재학습 사용 여부 등)는 각 제공자의 개인정보 처리방침 및 이용약관에 따릅니다.

7. 개인정보의 안전성 확보조치

확장프로그램은 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취합니다.

  • 로컬 저장 원칙: 요약/채팅/설정/그래프 등 대부분의 데이터는 이용자 브라우저 로컬에만 저장
  • 전송 구간 보호: AI 제공자 API 통신 시 HTTPS 사용
  • 접근 제한: 브라우저/확장프로그램 권한 범위 내에서만 접근하도록 설계
  • 최소 권한: 기능 제공에 필요한 범위에서 권한을 사용(예: 본문 추출, 선택 텍스트 처리 등)

※ 이용자의 PC/브라우저 환경(악성코드 감염, 계정 탈취 등)으로 인한 위험은 이용자 환경에 좌우될 수 있으므로, 운영체제/브라우저 보안 업데이트 및 계정 보안을 권장합니다.

8. 개인정보 자동 수집 장치의 설치·운영 및 거부

확장프로그램은 자체적으로 "쿠키"를 발급·운영하지 않습니다. 다만, 웹페이지 본문 추출/요약 등 기능 수행 과정에서 사용자가 방문 중인 웹사이트가 쿠키 등 자동 수집 장치를 사용할 수 있으며, 이는 해당 웹사이트의 정책에 따릅니다.

또한 AI 제공자의 SDK/쿠키를 확장프로그램이 임의로 설치·운영하지 않습니다(직접 API 호출 방식).

9. 정보주체의 권리·의무 및 행사방법

이용자는 언제든지 다음과 같은 방법으로 본 확장프로그램의 로컬 저장 데이터를 관리할 수 있습니다.

  • 설정값/기록 삭제: 확장프로그램 내 제공되는 설정/관리 화면에서 삭제(또는 초기화) 기능을 이용
  • 확장프로그램 제거: Chrome 확장프로그램 관리 화면에서 제거 시 로컬 저장 데이터가 삭제될 수 있음
  • AI 전송 범위 통제: "@페이지" 기능 미사용, 선택 텍스트 최소화 등으로 전송 데이터 범위 조절

※ AI 제공자에게 전송된 데이터에 대한 권리 행사는 각 제공자의 정책 및 절차에 따라 별도로 진행해야 할 수 있습니다.
